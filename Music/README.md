# Tendenciosa

## Descrição

Projeto desenvolvido em Java para reproduzir trechos da música **Tendenciosa** de **Veigh** utilizando um efeito de digitação no terminal.

Cada caractere é exibido gradualmente com um pequeno intervalo de tempo (`Thread.sleep()`), simulando uma máquina de escrever.

---

## Funcionalidades

- Efeito de digitação caractere por caractere.
- Pausa entre as frases.
- Utilização de arrays e laço `for`.
- Método reutilizável para escrever qualquer texto no terminal.

---

## Exemplo

**Saída**

```text
Já pensou se é o G-Hard com a peça na cinta?
Já pensou se é o Koda com a peça na cinta?
Já pensou se é o Nagalli com a peça na cinta?
Já pensou se é o Honaiser com a peça na cinta?
Já pensou se é o Toledo com a peça na cinta?

Já pensou se é uma cinta com a peça intocada?
Já pensou se eu aperto essa porra pelo olho mágico, e você tá sentado na sala?

VEIGH BABY UUUUUUUUUUUUUUUUUUUUUUUUU
```

---

## Conceitos praticados

- Métodos
- Arrays
- Laço `for`
- Manipulação de `String`
- `Thread.sleep()`
- Tratamento de exceções (`InterruptedException`)

---

## Como executar

1. Clone o repositório.
2. Abra o projeto em uma IDE Java (VS Code, IntelliJ ou NetBeans).
3. Execute a classe `Tendenciosa`.
