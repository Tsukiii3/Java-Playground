# Java Playground

Repositório destinado a pequenos projetos, experimentos e ideias desenvolvidas em Java.

O objetivo é praticar conceitos da linguagem criando aplicações simples e criativas, explorando recursos como manipulação de strings, efeitos no terminal, estruturas de dados, animações em console e outros desafios.

## Projetos

- Music
  - Tendenciosa – Reprodução de trechos da música com efeito de digitação no terminal.

Mais projetos serão adicionados conforme novos experimentos forem sendo desenvolvidos.
